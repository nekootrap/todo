7. Консольный To-Do List (Список дел)
Цель: Научиться работать со списками, выполнять CRUD-операции (Create, Read, Update, Delete), сохранять данные в файл.
**Требования:**
*   При запуске программа загружает список дел из файла `todo.txt` (каждая задача на новой строке, формат: `[ ] Купить молоко` или `[x] Прочитать книгу`).
*   Отображает меню:
    ```
    1. Показать все задачи
    2. Добавить задачу
    3. Отметить задачу как выполненную
    4. Удалить задачу
    5. Выход
    ```
*   При показе задач выводится их номер, статус `[ ]` или `[x]` и текст.
*   При добавлении/изменении/удалении задачи изменения сразу сохраняются в файл.
*   **Расширение:** Добавить дату создания задачи, возможность редактирования текста задачи, сортировку по статусу.
**Критерии оценки:** Полнота CRUD-функционала, надежное сохранение данных в файл, удобный интерфейс.
